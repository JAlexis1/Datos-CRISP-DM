# Datos-CRISP-DM
Proyecto de análisis de un problema de negocio basado en datos, siguiendo las fases de la metodología CRISP-DM
Basados en un dataset de diferentes transacciones de tarjetas de crédito, buscando desarrollar un modelo para detectar fraudes.<br>
